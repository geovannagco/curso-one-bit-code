High-Order Function - Exemplos

Map

A função map permite aplicar uma determinada operação a cada elemento de um array e retornar um novo array com os resultados. Ela recebe uma função como argumento, que é aplicada a cada elemento do array original.

Filter

A função filter permite criar um novo array contendo apenas os elementos que atendem a uma determinada condição. Ela também recebe uma função como argumento, que é usada para verificar se cada elemento satisfaz a condição.

Reduce

A função reduce permite reduzir os elementos de um array a um único valor. Ela recebe uma função de redução como argumento, que é aplicada a cada elemento do array e acumula um valor final.

Sort

A função sort permite ordenar os elementos de um array com base em uma determinada condição. Ela recebe uma função de comparação como argumento, que determina a ordem dos elementos.
