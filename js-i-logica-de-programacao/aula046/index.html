Retorno de Função

O retorno de uma função é o valor que a função "entrega" quando é executada. O retorno é especificado usando a palavra-chave 'return' seguida pelo valor que você deseja retornar.

Característica sobre retorno de função:

  • 1. Valor de Retorno: Uma função pode retornar qualquer valor, incluindo números, strings, objetos, arrays, booleanos e até mesmo outras funções. O valor de retorno é o resultado da computação realizada dentro da função. 
    •         
        function somar(a, b) {
          return a + b;
        }

        let resultado = somar(2, 3);
        console.log(resultado); // Output: 5
  • 2. Finalização da execução: Quando uma função encontra a declaração 'return', ela encerra sua execução imediatamente e retorna o valor especificado. Qualquer código abaixo da instrução 'return' não será executado. 
    •         
          function verificarPar(num) {
            if (num % 2 === 0) {
              return true;
            }
            console.log('Esta linha não será executada se o número for par.')
          }

          let isPar = verificarPar(4);
          console.log(isPar); // Output: true
  • 3. Funções sem retorno explícito: Se uma função não tiver uma instrução 'return' ou se a instrução 'return' não especificar um valor, a função retornará 'undefined' por padrão (default). 
    •         
          function saudar() {
            console.log('Olá!');
            // Não há instrução de retorno
          }

          let resultado = saudar();
          console.log(resultado); // Output: undefined
  • 4. Uso de múltiplos retornos: Uma função pode ter várias instruções 'return' ao longo de seu corpo, dependendo das condições. Quando a primeira instrução 'return' é encontrada e executada, a função é encerrada e retorna o valor correspondente. As instruções 'return' subsequentes não serão alcançadas. 
    •           function verificarIdade(idade) {
            if (idade < 0) {
              return 'Idade Inválida';
            } else if (idade < 18) {
              return 'Menor de idade';
            } else {
              return 'Maior de idade';
            }

          }
          
          console.log(verificarIdade(25)); //Output: Maior de idade
