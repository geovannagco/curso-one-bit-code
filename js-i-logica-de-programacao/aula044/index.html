Funções

Funções são blocos de código que podem ser definidos e reutilizados para executar tarefas específicas. Elas ajudam a organizar o código, tornando-o mais modular e legível. As funções podem receber valores de entrada, chamados de argumentos, e retornar valores de saída.

Sintaxe:


      function nomeDaFuncao(entrada) {
        // procedimentos...
        return saida
      }
