Arrays (Vetores)

  • Em JavaScript um array é uma estrutura de dados que permite armazenar vários valores em uma única variável. Esses valores podem ser de qualquer tipo, como números, strings, objetos e até mesmo outros arrays.
  • A principal característica de um array é que os elementos são organizados de forma sequencial, ou seja, eles têm uma ordem específica.
  • Cada elemento é associado a um índice, que começa em 0 para o primeiro elemento, 1 para o segundo elemento e assim por diante.

Sintaxe:

 const array_name = [item1, item2, ...]

É uma prática comum declarar arrays com a palavra-chave const.

Possuem funções para vários casos de uso:

  • enfileirar e desenfileirar
  • empilhar e desempilhar
  • achar o índice de um valor
  • cortar e concatenar
  • etc

Além disso, podemos obter informações sobre o array usando propriedades e métodos específicos. Alguns comuns são:

  • length: propriedade que retorna o número de elementos no array
  • push(): método que adiciona um novo elemento no final do array
  • pop(): método que remove o último elemento do array
  • indexOf(): método que retorna o índice de um elemento específico no array
