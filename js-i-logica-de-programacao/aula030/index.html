Estruturas de Repetição: For

Os loops são úteis para executar o mesmo código várias vezes, cada vez com um valor diferente.

A instrução for cria um loop com 3 expressões opcionais.

Sintaxe:

      
        for (expressão 1 (inicialização); expressão 2 (condição); expressão 3 (finalização)) {
          // bloco de código a ser executado
        }
  • A expressão 1 é executada (uma vez) antes da execução do bloco de código
    1. Normalmente, a primeira expressão é usada para inicializar a variável usada no loop
    2. A expressão 1 não é obrigatória
    3. Muitos valores podem ser inicializados, desde que separados por vírgula
  • A expressão 2 define a condição para executar o bloco de código
    1. A expressão 2 também é opcional
    2. Se a expressão 2 retornar true, o loop começará novamente. Se retornar false o loop terminará
    3. Atenção! Se a expressão 2 for omitida, deverá ser fornecida uma quebra dentro do loop. Caso contrário, o loop nunca terminará.
  • A expressão 3 é executada (todas as vezes) após a execução do bloco de código
    1. Frequentemente a expressão 3 é usada para incrementar o valor da variável inicial
    2. Também é opcional
    3. Poderá conter incrimento i++ ou decremento i--
