Funções Recursivas

Em JavaScript, funções recursivas são aquelas que chamam a si mesmas durante sua execução. Essas funções são úteis quando precisamos resolver um problema de forma iterativa, dividindo-o em subproblemas que podem ser resolvidos utilizando a mesma função.

Exemplo:

      
        function factorial(n) {
          // Caso base: quando n é igual a 0 ou 1, o fatorial é 1
          if (n === 0 || n === 1) {
            return 1;
          }

          // Chamada recursiva: n * (n - 1)!
          return n * factorial(n - 1);
        }

        console.log(factorial(5)); // Retorna 120 (5! = 5 * 4 * 3 * 2 * 1)

No exemplo, a função 'factorial' é definida para calcular o fatorial de um número 'n'. Ela verifica se 'n' é igual a 0 ou 1, o que são os casos base. Se for esse o caso, a função retorna 1, pois o fatorial de 0 ou 1 é 1. Caso contrário, a função chama a si mesma com 'n - 1' e multiplica o resultado pelo valor de 'n'. Esse processo é repetido até que 'n' seja reduzido a 0 ou 1, e então os resultados são retornado em cascata para calcular o resultadso final.
