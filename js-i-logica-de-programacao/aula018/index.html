JavaScript if, else, e else if

Em JS, temos as seguintes declarações condicionais:

  • Use if para especificar um bloco de código a ser executado se uma condição especificada for verdadeira.
  • Use else para especificar um bloco de código a ser executado se a mesma condição for falsa.
  • Use else if para especificar uma nova condição para testar se a primeira condição for falsa.
  • Use switch para especificar muitos blocos alternativos de código a serem executados.

If

Sintaxe:

        
          if (condition) { 
            // código a ser executado se a condição for verdadeira 
          }

Else

Sintaxe:

        
          if (condition) { 
            // código a ser executado se a condição for verdadeira 
          } else { 
            // código a ser executado se a condição for falsa 
          }

Else if

Sintaxe:

        
          if (condition) {
            // código a ser executado se a condição for verdadeira
          } else if (condition2) {
            // código a ser executado se a primeira condição for falsa e 
            a segunda condição for verdadeira
          } else {
            // código a ser executo se a primeira e a segunda condição forem falsas
          }
