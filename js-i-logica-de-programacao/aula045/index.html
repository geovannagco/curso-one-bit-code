Parâmetros de uma Função

Os parâmetros de uma função são os valores que podem ser recebidos quando a função é chamada. Eles permitem que você passe dados para a função, que podem ser usados internamente dentro do bloco de código da função.

Características dos parâmetros:

  • 1. Definição de parâmetros: os parâmetros são definidos na declaração da função entre parênteses, separados por vírgulas. Cada parâmetro é composto por um nome único que você escolhe.

    • Sintaxe

            
          function saudar(nome, idade) {
            // código da função
          }
  • 2. Número de argumentos: Ao chamar a função, você precisa fornecer um argumento correspondente para cada parâmetro definido. O número de argumentos e prâmetros deve ser igual. Caso contrário, os parâmetros restantes serão undefined. 
    •         
          saudar('João', 25);
  • 3. Valores padrão: Você pode atribuir um valor padrão a um parâmetro, caso nenhum argumento correspondente seja fornecido ao chamar a função. Isso é feito usando o operador de atribuição ('=') após a declaração do parâmetro. 
    •         
          function saudar(nome = 'Visitante') {
            console.log('Olá, ' + nome + '!');
          }

          saudar(); // Output: Olá, Visitante!
          saudar('João'); /// Output: Olá, João!
  • 4. Acesso aos parâmetros: Dentro do corpo da função, você pode usar parâmetros como variáveis locais, referenciado-os pelo nome que você atribuiu a eles. 
    •         
          function saudar(nome, idade) {
            console.log('Olá, ' + nome + '! Você tem ' + idade + ' anos.');
          }
