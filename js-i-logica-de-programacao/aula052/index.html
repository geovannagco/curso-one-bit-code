Funções Anônimas

Uma função anônima em JavaScript é uma função que não possui um nome atribuído. Em vez de definida com um nome como uma função regular, ela é criada como uma expressão e pode ser atribuida a uma variável ou passada como argumento para outra função.

Exemplo:

      
        const soma = function(a, b) {
          return a + b;
        }

        console.log(soma(2, 3)); // Retorna 5

Funções anônimas são frequentemente usadas em JavaScript para várias finalidades, como:

  • Callbacks: São passadas como argumentos para outras funções para serem chamadas em algum momento posterior, geralmente quando um evento ocorre ou quando uma operação assíncrona é concluída. Um exemplo comum é o uso de funções anônimas em funções 'setTimeout' ou 'addEventListener'. 
              
            setTimeout(function()) {
              console.log('Tempo Expirado!');
            }, 1000);
  • Closures: Funções anônimas podem ser usadas dentro de outras funções para criar um escopo encapsulado e preservar o estado de variáveis. Isso permite criar funções que carregam consigo um contexto específico. 
              
            function contador() {
              let count = 0;

              return function() {
                count++;
                console.log(count);
              }
            }

            const incrementar = contador();
            incrementar(); // Retorna 1
            incrementar(); // Retorna 2
  • IIFE (Immediately Invoked Expression): Funções anônimas também podem ser usadas para criar expressões de função invocadas imediatamente. Elas são úteis para evitar poluir o escopo global e para criar um escopo isolado para variáveis temporárias. 
              
            (function() {
              const nome = 'João';
              console.log('Olá, ${nome}!')
            })();
