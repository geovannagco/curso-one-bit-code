High-Order Functions

As high-order functions (funções de alta ordem) são funções que podem receber outras funções como argumentos e/ou retornar funções como resultado. Isso significa que as funções em JavaScript são tratadas como valores de primeira classe, assim como números ou strings.

As high-order functions permitem uma programação mais flexível e poderosa, tornando o código mais modular e fácfil de ler. Com elas, você pode escrever funções que são mais genéricas e reutilizáveis, já que podem receber comportamentos específicos através de funções passadas como argumentos.

Um exemplo clásico de high-order function em Js é a função 'map()'. Ela recebe uma função como argumento e a aplica a cada elemento de um array, retornando um novo array com os resultados.

Exemplo:

      
        const numeros = [1, 2, 3, 4, 5];

        const aoQuadrado = numeros.map(function(numero) {
          return numero ** 2;
        });

        console.log(aoQuadrado); // [1, 4, 9, 16, 25]

Nesse exemplo, a função 'map()' recebe uma função anônima como argumento, que eleva cada número ao quadrado. A 'map()' aplica essa função a cada elemento do array 'numeros' e retorna um novo array com os valores resultantes.
