Estruturas Condicionais: Switch

  1. A expressão switch é avaliada uma vez.
  2. O valor da expressão é comparado com os valores de cada caso.
  3. Se houver uma correspondência, o bloco de código associado será executado.
  4. Se não houver correspondência, o bloco de código padrão será executado.

Sintaxe:

switch (expressao) {
        case 'a':
        //...
        case 'b':
        //...
          break
        case 'c':
        //...
        default:
        //...
    }
