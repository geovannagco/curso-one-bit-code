Escopo

O escopo em funções refere-se à visibilidade e acessibilidade das variáveis e funções dentro de uma função específica. Em JavaScript, existem dois principais tipos de escopo em funções: o escopo local e o escopo global.

  • 1. Escopo local: Quando você declara uma variável dentro de uma função, ela é acessível apenas dentro dessa função. Essa variável tem um escopo local, o que significa que ela não pode ser acessada fora da função. 
    •         
          function minhaFuncao() {
            var x = 10; // Variável local
            console.log(x); // Output: 10
          }

          minhaFuncao();
          console.log(x); // Erro: x não está definido (x está fora do escopo local)
  • 2. Escopo global: As variáveis declaradas fora de qualquer função têm escopo global, o que significa que podem ser acessadas de qualquer lugar do seu código. 
    •         
          var x = 10; // Variável global

          function minhaFuncao() {
            console.log(x); // Output: 10
          }
